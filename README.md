# AI-COMPANION
AI-powered mental health platform with chatbot, mood tracking, meditation, music therapy, and real-time yoga posture detection.
# AI Companion - Mental Health Platform

## 🚀 Features
- AI Therapist (CBT-based chatbot)
- Mood Tracking
- Meditation & Music Therapy
- Yoga with Posture Detection (MediaPipe)

## 🛠 Tech Stack
- Frontend: React.js + Vite
- Backend: Node.js / Python
- Database: MongoDB / SQLite

## ⚙️ Setup Instructions
1. Clone repo
2. Install dependencies
3. Run frontend & backend
